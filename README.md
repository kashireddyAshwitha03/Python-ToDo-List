# Python To-Do List

## Project Overview
This is a simple To-Do List application developed using Python as part of the DecodeLabs Industrial Training Kit - Project 1.

## Features
- Add new tasks
- View all tasks
- Menu-driven interface
- User-friendly console output

## Technologies Used
- Python
- Lists
- Loops
- Conditional Statements
- User Input

## How to Run
1. Download the project files.
2. Open a terminal in the project folder.
3. Run:

```python
python todo_list.py
```
